Installation
⚠️

First-time Gatekeeper prompt. Because Table Read is distributed outside the Mac App Store, macOS will block it the first time you try to open it. This is normal for indie apps — follow the steps below to allow it. You'll only need to do this once.

  1. 1
    Open the downloaded TableRead-0.1.0.dmg file.
  2. 2
    Drag Table Read into the Applications folder shortcut in the window.
  3. 3
    Open Applications in Finder and double-click Table Read.
    macOS will show: "Table Read cannot be opened because it is from an unidentified developer." Click Done to dismiss it for now.
  4. 4
    Open System Settings → Privacy & Security. Scroll down until you see a message about Table Read being blocked. Click Open Anyway.
  5. 5
    Enter your Mac password or use Touch ID to confirm. Table Read will open — and won't ask again.

What it does
🎭

Full cast of voices

Every character gets a unique voice. macOS system voices are built in. Kokoro (free, local) and OpenAI TTS are optional upgrades.

📄

PDF to audio

Drop in any standard screenplay or play PDF. Table Read parses dialog, scenes, and stage directions automatically.

✏️

Editable parse

Review every line, fix mis-attributed dialog, rename speakers, or remove noise. Corrections are saved and re-applied next time.

🔒

Fully private

Runs 100% on your Mac. Nothing leaves your machine unless you choose to use OpenAI voices.

How it works
1

Import a script PDF

Click "Choose PDF" or drag a file in. Table Read parses all characters, dialog, and scenes in seconds.

2

Review & correct

Inspect every line. Fix any mis-attributed dialog or mark lines as noise. Corrections persist across re-imports.

3

Cast your voices

Assign a voice to each character. Install Kokoro (free, local) or connect OpenAI for the most natural results.

4

Generate audio

Select scenes and hit Generate. Each scene becomes a standalone .m4a file ready for playback or editing.
